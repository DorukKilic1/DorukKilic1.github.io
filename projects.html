Q38: Probability triangle condition Answer: 1/4. Explanation: Three random values between 0 and 1 form a triangle only when each is smaller than the sum of the other two, which happens 25% of the time.

Q39: Probability obtuse triangle Answer: About 0.52. Explanation: Valid random triangles tend to be obtuse slightly more than half the time.

Q40: Reverse digits Answer: "123456789". Since each extracted digit is added to the front, the number becomes reversed.

Q41: Increment confusion Answer: 24. Mixed pre/post increments cause one operand to use old value, one new.
