2.1 – 2.10: Algorithms, Loops, and Strings

  • 2.1 Algorithms, Selection, and Iteration
    • Algorithm → Instructions designed to accomplish a task.
    • Selection → Picks between options using if, if-else.
    • Iteration → Looping using for or while.
  • 2.2 Building Algorithms Using Selection and Iteration
    • if(score >= 60) { System.out.println("Pass"); } else { System.out.println("Fail"); }
    • Decides if score is passing.
    • for(int i = 1; i <= 5; i++) { System.out.println(i); }
    • Prints numbers 1–5 individually.
  • 2.3 Applying the Accumulator Pattern
    • Initialize accumulator before loop.
    • Update inside loop.
    • Use result after loop.
    • int sum = 0;
      for(int i = 1; i <= 100; i++) { sum += i; }
      double average = (double)sum / 100;
  • 2.4 Loops by Input
    • Scanner scan = new Scanner(System.in);
      int number = scan.nextInt();
      int sum = 0, count = 0;
      while(number != -1) {
        sum += number;
        count++;
        number = scan.nextInt();
      }
      double average = (double)sum / count;
    • Loops until user enters -1, calculates average.
  • 2.5 Using if Statements in Loops for Min/Max Detection
    • int min = Integer.MAX_VALUE;
      int max = Integer.MIN_VALUE;
      for(int i = 0; i < n; i++) {
        int num = (int)(Math.random() * 100);
        if(num < min) min = num;
        if(num > max) max = num;
      }
    • Tracks smallest and largest random values.
  • 2.6 Checking for Divisibility
    • if(num % 2 == 0) → num is even.
    • if(num % divisor == 0) → num divisible by divisor.
  • 2.7 Extracting and Matching Digits
    • while(number > 0) {
        int lastDigit = number % 10;
        if(lastDigit == digit) return true;
        number /= 10;
      }
      return false;
    • Checks if a number contains a specific digit.
  • 2.8 Counting Frequency
    • int count = 0;
      for(int i = 1; i <= 100; i++) {
        if(isPrime(i)) count++;
      }
    • Counts number of primes between 1 and 100.
  • 2.9 Writing Algorithms Using Selection and Repetition
    • Common patterns include:
    • Input-controlled loops
    • Divisibility checks
    • Min/Max detection
    • Digit matching
    • Frequency counting
    • Accumulator pattern
  • 2.10 Working with Strings in Algorithms
    • 2.10.1 Search and Replace Characters
      • int index;
        while(message.indexOf("1") >= 0) {
          index = message.indexOf("1");
          message = message.substring(0, index) + "x" + message.substring(index + 1);
        }
      • Replaces all "1"s with "x".
    • 2.10.2 Traversing a String Using a for Loop
      • for(int i = 0; i < s.length(); i++) {
          String ch = s.substring(i, i + 1);
        }
      • Accesses each character in a string.
    • 2.10.3 Counting Specific Characters
      • int count = 0;
        for(int i = 0; i < s.length(); i++) {
          if(s.substring(i, i + 1).equals("e")) count++;
        }
      • Counts how many times "e" appears.
    • 2.10.4 Reversing a String
      • String sReversed = "";
        for(int i = 0; i < s.length(); i++) {
          sReversed = s.substring(i, i + 1) + sReversed;
        }
      • Creates a reversed version of the string.
    • 2.10.5 Replacing a Word in a String
      • while(message.indexOf("Ilgaz") >= 0) {
          int index = message.indexOf("Ilgaz");
          message = message.substring(0, index) + "Rüzgar" + message.substring(index + 5);
        }
      • Changes all occurrences of "Ilgaz" to "Rüzgar".
