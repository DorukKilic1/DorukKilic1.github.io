CSAwesome Notes (Rephrased)

Simplified notes from Unit 1 of CSAwesome.

Unit 1

  • 1.1 Introduction: Algorithm = Steps, Sequence = Order, System.out.println() prints, ; ends, Compiler = syntax check, Logic error = wrong output, Runtime error = execution error.
  • 1.2 Variables: Variable = memory, Primitive = int/double/boolean, Reference = objects, String = words.
  • 1.3 Expressions: println new line, Literals = fixed, \n, \", \\, Ops = + - * /, int ÷ int = int, % = remainder, Order = * / % before + -, ÷0 = error.
  • 1.4 Assignment: = assigns, Scanner = input.
  • 1.5 Casting: (int)/(double) convert, double→int cuts decimals, int in double widened, Rounding = (int)(x+0.5), Integer.MAX/MIN, Overflow = wrap.
  • 1.6 Compound Ops: x++ add 1, x*=y = x=x*y.
  • 1.7 APIs: Library = premade classes, API = methods/fields, Package = grouped classes.
  • 1.8 Docs: Comments // or /* */, Precondition = true before, Postcondition = true after.
  • 1.9 Methods: Method = named block, Params = header vars, Args = actual values, Call by value = copy, Overload = same name diff params.
  • 1.10 Class Methods: Static = class-level, Syntax Class.method(), Void = no return.
  • 1.11 Math: Math.abs(), Math.pow(), Math.sqrt(), Math.random().
  • 1.12 Objects: Class = template, Object = instance, Attributes = vars/methods, All extend Object.
  • 1.13 Constructors: new = create, Constructor = same name, Overload = multiple versions.
  • 1.14 Instance Methods: obj.method(), Null → NullPointerException.
  • 1.15 Strings: String = immutable object, + or += concat, Methods length(), substring(), indexOf(), equals(), compareTo(), Index starts 0.
  • 1.16 Summary A: Algorithms, IDE, errors, int/double/boolean, assignment, printing, precedence, casting, overflow, round-off, compound ops.
  • 1.20 Summary B: Objects/classes, methods, constructors, Math, Strings, Wrapper classes.
