CSAwesome Notes

1.1 Programs and Errors
  • Algorithm = Steps to solve.
  • Sequence = Order of steps.
  • System.out.println() = Prints text.
  • ; = Ends a line of code.
  • Compiler = Finds syntax mistakes.
  • Logic error = Wrong result.
  • Runtime error = Error while running.
1.2 Variables and Types
  • Variable = Memory storage.
  • Primitive = int, double, boolean, etc.
  • Reference = Objects (like String).
  • String = Text.
1.3 Printing and Expressions
  • println = Prints on new line.
  • Literal = Fixed value.
  • \n = New line, \" = Quote, \\ = Backslash.
  • Operators = +, -, *, /.
  • int ÷ int = int result unless double involved.
  • % = Remainder.
  • Order → *, /, % before +, -.
  • Divide by 0 = Error.
1.4 Assignment and Input
  • Scanner = Reads user input.
1.5 Casting and Limits
  • (int), (double) = Convert types.
  • double → int = Cuts decimals.
  • int in double math = Becomes double.
  • Round = (int)(x + 0.5).
  • Integer.MAX/MIN = Biggest, smallest int.
  • Overflow = Wraps around.
1.6 Shortcuts
  • x++ = Add 1.
  • x *= y = Same as x = x * y.
1.7 Libraries
  • Library = Prebuilt classes.
  • API = Methods and fields.
  • Package = Class group.
1.8 Docs and Conditions
  • Comments = // or /* */.
  • Precondition = Must be true before.
  • Postcondition = True after.
1.9 Methods
  • Method = Named code block.
  • Parameters = In header.
  • Arguments = Values passed.
  • Call by value = Copy sent.
  • Overload = Same name, different params.
1.10 Static Methods
  • Static = Class level.
  • Syntax = ClassName.method().
  • Void = No return.
1.11 Math Class
  • Math.abs() = Absolute value.
  • Math.pow(x, y) = Power.
  • Math.sqrt() = Square root.
  • Math.random() = 0.0–1.0 random.
1.12 Objects
  • Class = Template.
  • Object = Instance.
  • Attributes = Vars + methods.
  • All classes extend Object.
1.13 Constructors
  • new = Makes object.
  • Constructor = Same as class name.
  • Overload = Different params.
1.14 Instance Methods
  • Syntax = obj.method().
  • Null object → NullPointerException.
1.15 Strings
  • String = Object, immutable.
  • +, += = Concatenate.
  • methods = length(), substring(), indexOf(), equals(), compareTo().
  • Index starts at 0.
1.16 Summary A
  • Algorithms, IDE, errors.
  • int, double, boolean.
  • Assignment, printing, operator order.
  • Casting, overflow, rounding.
  • Shortcut ops.
1.20 Summary B
  • Objects + classes.
  • Static vs instance methods.
  • Constructors.
  • Math methods.
  • Strings.
  • Wrapper classes.
