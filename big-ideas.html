Big Ideas Notes

These notes summarize key Java programming concepts and coding fundamentals.

  • Java: A programming language used worldwide to create software.
  • Algorithms: Step-by-step processes to complete tasks or solve problems.
  • Compiler: Error checker. Syntax errors are reported when found.
  • Logic errors: Most likely math errors.
  • Comments: // for single-line, /* */ for multi-line.
  • Variable: Memory location storing a value that can change during runtime.
  • Code examples:
    • System.out.println()
    • Double example: 6.353535
    • Int example: 4
    • Boolean: true or false
    • String: Words
  • To declare a variable: specify type, a space, the name, and end with ;.
  • Printing:
    • System.out.print ```
