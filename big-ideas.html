Big Ideas Notes

These notes summarize key Java programming concepts and coding fundamentals.

  • Java: A programming language used worldwide to create software.
  • Algorithms: Step-by-step processes to complete tasks or solve problems.
  • Compiler: Error checker. Syntax errors are reported when found.
  • Logic errors: Most likely math errors.
  • Comments: // for single-line, /* */ for multi-line.
  • Variable: Memory location storing a value that can change during runtime.
  • Code examples:
    • System.out.println()
    • Double example: 6.353535
    • Int example: 4
    • Boolean: true or false
    • String: Words
  • To declare a variable: specify type, a space, the name, and end with ;.
  • Printing:
    • System.out.println(value) prints value + new line
    • System.out.print(value) prints value without new line
  • Data Types: Integers, doubles, booleans.
  • Documentation: Important to make code clear. Example: instruction manual, API.
  • Java Keywords:
    • public → accessible anywhere
    • class → blueprint for objects
    • static → belongs to class, not object
    • void → method that doesn’t return a value
    • main → starting point of program
  • API: Documentation that explains what a library can do and how to use it.
  • Incrementing a variable: a = a + 1;, a += 1;, a++;
  • User input: Scanner in = new Scanner(System.in);

CSAwesome Notes

  • 1.1 Programs and Errors
    • Algorithm = Steps to solve.
    • Sequence = Order of steps.
    • System.out.println() = Prints text.
    • ; = Ends a line of code.
    • Compiler = Finds syntax mistakes.
    • Logic error = Wrong result.
    • Runtime error = Error while running.
  • 1.2 Variables and Types
    • Variable = Memory storage.
    • Primitive = int, double, boolean, etc.
    • Reference = Objects (like String).
    • String = Text.
  • 1.3 Printing and Expressions
    • println = Prints on new line.
    • Literal = Fixed value.
    • \n = New line, \" = Quote, \\ = Backslash.
    • Operators = +, -, *, /.
    • int ÷ int = int result unless double involved.
    • % = Remainder.
    • Order → *, /, % before +, -.
    • Divide by 0 = Error.
  • 1.4 Assignment and Input
    • Scanner = Reads user input.
  • 1.5 Casting and Limits
    • (int), (double) = Convert types.
    • double → int = Cuts decimals.
    • int in double math = Becomes double.
    • Round = (int)(x + 0.5).
    • Integer.MAX/MIN = Biggest, smallest int.
    • Overflow = Wraps around.
  • 1.6 Shortcuts
    • ++ = Increase by 1.
    • += = Add to variable.
  • 1.7 Methods
    • Method = Function with name.
    • Parameters = Input to method.
    • Return = Output from method.

Additional Java Notes

Java Keywords: public, class, static, void, main
Comments: Use // for single-line, /* ... */ for multi-line comments.
Variables: Declared as type name; and initialized as name = value;.
Data types: int, double, boolean, String.
Printing: Use System.out.println() to print with newline.
Input: Use Scanner for user input.
Increment: a++, a += 1;, or a = a + 1;.
API Documentation: Explains how to use libraries and classes.
Common errors: Syntax errors (compile-time), logic errors, runtime errors.
